HTTP Status Pages

ステータスコードを選ぶと、パソコンはよく分からないけれど専門用語だけは妙に詳しいおばあちゃんが、成功・転送・焦り・サーバー疲れまで表情豊かに説明します。

1xx Informational

100👍
Continue
クライアントは、このままリクエストの送信を続けてかまいません。
101🔄
Switching Protocols
サーバーは、クライアントが要求した通信プロトコルへ切り替えます。
102🛠️
Processing
サーバーはリクエストを受け取り、処理を続けています。
103💡
Early Hints
最終的なレスポンスの前に、関連リソースを先に読み込むための情報を返します。
104⏫
Upload Resumption Supported
サーバーは、途中で止まったアップロードの再開に対応しています。

2xx Success

200✅
OK
リクエストは正常に処理されました。
201🆕
Created
リクエストは成功し、新しいリソースが作成されました。
202📩
Accepted
リクエストは受け付けられましたが、処理はまだ完了していません。
203🗞️
Non-Authoritative Information
返された情報は、元のサーバーから直接得られたものではありません。
204📭
No Content
リクエストは成功しましたが、返す本文はありません。
205🧹
Reset Content
リクエストは成功しました。クライアント側の入力画面などをリセットします。
206🧩
Partial Content
リソースの一部分だけを返しています。
207📋
Multi-Status
複数の処理結果をまとめて返しています。
208🔁
Already Reported
その情報は、すでに別の場所で報告済みです。
226🧮
IM Used
差分処理などを適用した結果を返しています。

3xx Redirection

300🧭
Multiple Choices
リクエストに対して、複数の候補があります。
301🏠
Moved Permanently
リソースは恒久的に別のURLへ移動しました。
302🚧
Found
リソースは一時的に別のURLにあります。
303👀
See Other
別のURLをGETメソッドで参照してください。
304♻️
Not Modified
リソースは前回取得したときから変更されていません。キャッシュを利用できます。
305🧱
Use Proxy
指定されたプロキシを使ってアクセスする必要があります。
306💤
Unused
以前は使われていましたが、現在は使われていないステータスコードです。
307↪️
Temporary Redirect
一時的に別のURLへリダイレクトします。HTTPメソッドは変更しません。
308🔀
Permanent Redirect
恒久的に別のURLへリダイレクトします。HTTPメソッドは変更しません。

4xx Client Error

400😵
Bad Request
リクエストの構文や内容に誤りがあり、サーバーは処理できません。
401🔐
Unauthorized
アクセスするには認証が必要です。
402💸
Payment Required
将来的な支払い処理のために予約されているステータスコードです。
403⛔
Forbidden
アクセス権限がないため、リクエストは拒否されました。
404🐈
Not Found
要求されたリソースは見つかりませんでした。
405🚫
Method Not Allowed
対象リソースでは、そのHTTPメソッドは使えません。
406🙅
Not Acceptable
クライアントが受け入れ可能な形式では、レスポンスを返せません。
407🧱🔐
Proxy Authentication Required
プロキシサーバーでの認証が必要です。
408⏰
Request Timeout
クライアントからのリクエスト送信が、制限時間内に完了しませんでした。
409⚔️
Conflict
リクエストの内容が、現在のリソースの状態と競合しています。
410👻
Gone
要求されたリソースは恒久的に利用できなくなっています。
411📏
Length Required
Content-Lengthヘッダーが必要です。
412📌
Precondition Failed
リクエストで指定された前提条件を満たしていません。
413🐘
Content Too Large
リクエスト本文が大きすぎるため、サーバーは処理できません。
414🐍
URI Too Long
リクエストURIが長すぎるため、サーバーは処理できません。
415📦
Unsupported Media Type
リクエスト本文のメディア形式に、サーバーは対応していません。
416🎯
Range Not Satisfiable
指定された範囲のデータを返せません。
417🤔
Expectation Failed
Expectヘッダーで求められた条件を満たせませんでした。
418🫖
Unused
現在は使われていないステータスコードです。
421🧭❌
Misdirected Request
リクエストは、処理できないサーバーに送られています。
422🧠
Unprocessable Content
リクエストの構文は正しいものの、内容の意味を処理できません。
423🔒
Locked
対象リソースはロックされています。
424🧱
Failed Dependency
依存している別の処理が失敗したため、このリクエストも処理できません。
425🐣
Too Early
リプレイ攻撃の危険があるため、サーバーはこのリクエストを処理しません。
426⬆️
Upgrade Required
別のプロトコルへアップグレードする必要があります。
428📋
Precondition Required
リクエストには前提条件を含める必要があります。
429🚦
Too Many Requests
短時間に送信されたリクエストが多すぎます。
431🧾
Request Header Fields Too Large
リクエストヘッダーが大きすぎるため、サーバーは処理できません。
451⚖️
Unavailable For Legal Reasons
法的な理由により、リソースを提供できません。

5xx Server Error

500💥
Internal Server Error
サーバー内部で予期しないエラーが発生しました。
501🧰
Not Implemented
サーバーには、リクエストを処理するための機能が実装されていません。
502🚪💥
Bad Gateway
ゲートウェイまたはプロキシが、上流サーバーから不正な応答を受け取りました。
503🚧
Service Unavailable
サーバーは一時的にリクエストを処理できない状態です。
504⏳
Gateway Timeout
ゲートウェイまたはプロキシが、上流サーバーから時間内に応答を受け取れませんでした。
505📜
HTTP Version Not Supported
サーバーは、リクエストで使われたHTTPバージョンに対応していません。
506🔁
Variant Also Negotiates
サーバーのコンテンツネゴシエーション設定に問題があります。
507🗄️
Insufficient Storage
サーバーは、リクエストを完了するために必要な保存領域を確保できません。
508🔄
Loop Detected
サーバーは、処理中に無限ループを検出しました。
510🧓
Not Extended
リクエストを処理するには、必要な拡張情報が不足しています。
511📶🔐
Network Authentication Required
ネットワークへアクセスするには認証が必要です。
